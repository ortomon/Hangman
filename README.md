# Hangman (Виселица)

Консольная игра «Виселица».  
Программа загадывает слово, игрок по буквам пытается его угадать.

## Как играть

1. Запустите приложение.
2. В главном меню:
    - введите `1`, чтобы начать новую игру
    - введите `2`, чтобы выйти
3. Игра загадывает слово и показывает его в виде `*`.
4. Введите одну букву.
    - если буква есть в слове — она откроется
    - если буквы нет — увеличится счётчик ошибок
    - если буква уже вводилась — ход не засчитывается
5. Игра продолжается, пока:
    - слово не будет угадано (победа)
    - или не будет допущено максимальное количество ошибок (поражение)

## Условия

- Вводится ровно одна буква русская буква
- Максимальное количество ошибок — 6

## Результат

- Победа — все буквы слова открыты
- Поражение — превышен лимит ошибок, игра показывает загаданное слово
